### API calls:
 
`
POST /api/v1/changelog
`
to store a new changelog

`
GET /api/v1/changelog
`
to get all changelogs

`
GET /api/v1/exampleBugfixes
`
to get bugfixes 

`
POST /api/v1/init
`
to initialize the changelogs table
